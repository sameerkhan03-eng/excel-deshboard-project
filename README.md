📊 **Vrinda Store Sales Dashboard (Excel Project)**

**🧾 Overview**

This project analyzes annual sales data of Vrinda Store using Microsoft Excel.
The dataset contains 30,000+ rows, which were cleaned, transformed, and used to build an interactive dashboard for business insights.

---

**🛠️ Tools Used**

- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie, Line)
- Slicers & Dashboard Design

---

**🔧 Data Preparation**

- Cleaned and processed 30,000+ rows of raw data
- Removed duplicates and handled missing values
- Standardized columns (dates, categories, gender, states,age group)
- Created calculated fields for analysis

---

**📌 Key Features**

- 📅 Monthly Sales Analysis
- 👥 Sales by Gender
- 📍 Top Performing States
- 📦 Order Status Distribution
- 🛒 Channel-wise Sales Contribution
- 🎯 Interactive Filters (Slicers)

---

**📊 Sample Insights**

- Women are more likely to buy compared to men (~65%)
- Maharashtra, Karnataka and Uttar Pradesh are the top 3 states (~35%)
- Adult age group (30–49 yrs) is max contributing (~50%)
- Amazon, Flipkart and Myntra channels are max contributing (~80%)

---

**🎯 Final Conclusion**

To improve Vrinda Store sales:

- Target women customers of age group (30–49 yrs)
- Focus on top states: Maharashtra, Karnataka, Uttar Pradesh
- Run ads, offers, and coupons on major channels:
  - Amazon
  - Flipkart
  - Myntra
